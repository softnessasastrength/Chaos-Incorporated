# The Lightweight Trench

*The most literal garment-scale execution of the ring's own design philosophy in outerwear — Medium tier. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the factory-ready manufacturing spec.*

The belted waist restates Starlink One's "soft shape held by hard architecture" at coat scale — the belt is functional structure, not a styling detail. See [`../README.md`](../README.md) for its place in the outerwear catalogue and the original concept note at [`../05-lightweight-trench.md`](../05-lightweight-trench.md), which this folder expands into full vision + manufacturing-brief format.

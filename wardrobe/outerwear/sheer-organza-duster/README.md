# The Sheer Organza Duster

*Pure Soft Light as outerwear — Experimental tier. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the factory-ready manufacturing spec.*

The cleanest possible expression of "soft light as outerwear" in the system — the entire garment is translucent, with structure and opacity living entirely in whatever's worn underneath. Governed by [`../../translucent-system.md`](../../translucent-system.md). Distinct from [`../kimono-duster/`](../kimono-duster/) (opaque/print). See [`../README.md`](../README.md) for its place in the outerwear catalogue and the original concept note at [`../07-sheer-organza-duster.md`](../07-sheer-organza-duster.md), which this folder expands into full vision + manufacturing-brief format.

## Status

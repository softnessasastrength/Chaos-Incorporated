# The Kimono Duster

*The purest "soft" register in the outerwear category — High tier. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the factory-ready manufacturing spec.*

Depends entirely on a fitted or structured layer underneath (camisole, fitted tee, corset top) for the manifesto's required soft/hard balance — this piece carries no structure of its own by design. Distinct from [`../sheer-organza-duster/`](../sheer-organza-duster/) (translucent Soft Light version). See [`../README.md`](../README.md) for its place in the outerwear catalogue and the original concept note at [`../03-kimono-duster.md`](../03-kimono-duster.md), which this folder expands into full vision + manufacturing-brief format.

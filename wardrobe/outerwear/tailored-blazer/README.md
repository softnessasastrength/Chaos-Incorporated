# The Lightweight Tailored Blazer

*The missing "hard architecture" layer for Soft Light and soft day dresses — High tier. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the factory-ready manufacturing spec.*

Fills a named gap between the hardware moto jacket (wrong occasion register) and the trench (too much coat) — dressier structure without either. Soft Heat stack: [`../../warm-season-system.md`](../../warm-season-system.md). See [`../README.md`](../README.md) for its place in the outerwear catalogue and the original concept note at [`../08-tailored-blazer.md`](../08-tailored-blazer.md), which this folder expands into full vision + manufacturing-brief format.

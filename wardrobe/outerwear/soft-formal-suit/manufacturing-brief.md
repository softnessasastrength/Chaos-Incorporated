# The Soft Formal Suit — Manufacturing Brief

**Document Type:** Technical Specification / Manufacturing Brief for pattern development and factory quoting
**Prepared for:** Formal suiting manufacturer capable of producing a matched blazer-plus-bottom set in one fabric lot
**Revision:** 1.0 (Draft — pending manufacturer confirmation of matched-fabric-lot process and trouser/skirt fit)
**Governing doc:** [`vision.md`](vision.md) — read for design rationale; this document is the buildable spec.

---

## 1. Product Overview

A matching two-piece suit: cropped-to-standard-short blazer (22–26") plus either a high-rise trouser or a midi pencil/A-line skirt, in stretch wool blend, crepe suiting, or formal ponte. Both pieces produced from a single matched fabric lot. Four colorways.

## 2. Pattern & Construction Specification

| Parameter | Target | Notes |
|---|---|---|
| Blazer length | 22–26" | Confirm on fit sample — same crop logic as the everyday tailored blazer |
| Blazer sleeve | To wrist bone, pushable | Confirm against live fit model |
| Bottom (trouser option) | High-rise, straight or soft wide-leg | Confirm hem length assumes a 2–2.5" heel — verify against actual heel height, not a flat assumption |
| Bottom (skirt option) | Midi pencil or soft A-line | Matching fabric to blazer |
| Closure | Single button, silver-tone (blazer); zip or hook, silver-tone (trouser) | |

## 3. Fabric Specification — Matched Lot Requirement

| Parameter | Target | Notes |
|---|---|---|
| Fabric | Stretch wool blend, crepe suiting, or formal ponte | **Both blazer and bottom must be cut from the same dye lot** — confirm this explicitly with the manufacturer, since suits ordered as separate components (rather than a true matched set) risk a visible color mismatch between pieces even when nominally the "same" colorway |
| Stretch | 2–5% elastane | Consistent across both pieces |
| Lining (blazer) | Lined or half-lined | Confirm sit-through-dinner comfort, same sit-test logic as the Soft Formal Midi Dress |

## 4. Hardware Specification

Silver buttons only on the blazer; silver zip/hook on the trouser. Confirm explicitly that gold crest buttons — a common formal-suiting default — are not used.

## 6. Starlink One Reference Match

Silver hardware only. No further reference-matching requirement — a quiet suit by design, so the ring leads (see `vision.md` Section 5).

## 7. Critical Technical Challenges — Open Items for Manufacturer Conversation

1. **Matched-lot fabric sourcing** — the single highest-priority open item; confirm the manufacturer's process guarantees blazer and bottom are cut from the same dye lot, not just the same nominal colorway ordered separately.
2. **Trouser hem length against actual heel height** — confirm hem is verified against a 2–2.5" heel sample, not assumed.
3. **Blazer sit-test comfort** — confirm lined/half-lined blazer holds up through a seated formal-dinner-length wear test.
4. **Gold-button default risk** — confirm silver hardware explicitly, since gold crest buttons are a common formal-suiting default.

## 8. Requested Next Step

A single matched set (blazer plus trouser, one colorway — Midnight Navy or Jet Black recommended) as the first fit sample, with fabric-lot matching and heel-dependent hem length both confirmed, before any additional colorway or the midi-skirt bottom option is produced.

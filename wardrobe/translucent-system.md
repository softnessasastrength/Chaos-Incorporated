# Translucent System — Soft Light

*Design doctrine for sheer and translucent pieces in Renn's wardrobe. Read [`wardrobe-manifesto.md`](wardrobe-manifesto.md) and [`color-system.md`](color-system.md) first. Soft Structure jellies (the material-translucent footwear cousin) live in [`shoes/jelly/README.md`](shoes/jelly/README.md).*

**Priority for the whole register:** Experimental  
**Phase:** Phase 2–3 seasonal / statement (docs built early; **do not source ahead of Phase 1 Must-Haves or core basics**)  
**Status:** Concept  

---

## Thesis: Translucent, Not Transparent

Starlink One is a soft shape held by hard architecture. Translucent clothing is the same idea applied to **light**:

- Soft field = sheer organza, washed chiffon, fine mesh, open lace — color and skin both visible  
- Hard architecture = lined zones, solid base layers, smocking, ribbon, silver hardware, defined waist  

**Never** one fully see-through head-to-toe with nothing engineered underneath. That is costume. Chaos Incorporated wants **controlled light** — a soft veil over deliberate structure.

The wardrobe already touches romance via lace *trim* and camisoles. This system is the next step: **translucency as a design register**, not an accident of flimsy fabric.

---

## Opacity Map (non-negotiable)

Every translucent garment must document which zones are which.

| Zone | Rule |
|---|---|
| **Bust** | Opaque, lined, or double-layered — always. Daywear does not negotiate this. |
| **Seat / crotch** | Opaque or fully covered by a designed underlayer (slip, bike short, lined panel). |
| **Torso midriff** | Translucent OK only *over* a solid base (cami, tank, seamless bra-friendly layer). |
| **Sleeves, yoke, hem tiers, outer duster** | Translucent OK — primary light fields. |
| **Full-body sheer outer** | Experimental only, and **only** when the underlayer is specified as part of the design (not "figure it out"). |

If a sourced piece is sheer at the bust with no lining and no intended solid base, it is **out of system** — do not force it into Soft Light by styling heroics.

## Palette as Light (not paint)

Translucent colorways **filter** light in palette tones rather than blocking as solids. Pull only from [`color-system.md`](color-system.md):

| Color | Translucent role |
|---|---|
| **Cloud Ivory** | Soft veil — quietest base; ring stays loudest |
| **Blush Mauve** | Romantic wash over skin |
| **Lavender Mist** | Soft cool purple filter |
| **Periwinkle** | Stronger cool filter — use as one field, not full look |
| **Sage** | Quiet botanical filter |
| **Starlink Emerald** | Hero translucent — same job as translucent jelly flat; **one big Emerald field per look max** |
| **Sterling Grey** | Cool shadow layer |
| **Midnight Navy / Jet Black** | Rare — sheer black/navy reads evening; Phase 3 statement only |

### One loud element

Same discipline as the rest of the system:

- Emerald translucent **top** *or* Emerald translucent **jelly** *or* Emerald as ring-led accent — not all three competing as large fields  
- The ring is always allowed; it is small and saturated. Large translucent Emerald elsewhere must stay quiet near the hands if the ring is meant to lead  

---

## Soft + Hard Pairing Rules

1. **Every soft translucent field needs one hard detail** — lined bust panel, smocked bodice, ribbon tie, silver button, chain-link clasp, or a solid underlayer that is part of the look.  
2. **Comfort is still the aesthetic** — no scratchy costume nylon lace at high skin-contact points; soft mesh or washed chiffon preferred.  
3. **64" frame** — sheer volume needs an exit strategy: cropped length, defined waist, knee-to-midi max on skirts, 3/4 sleeve or clear wrist on anything near the ring.  
4. **Layering is design, not rescue** — underlayers (ribbed tank, ribbon-tie cami, bike shorts, slip) are named in each item file.  
5. **Patina note** — translucency is optical. Fabrics should still be washable and livable; this is not a "precious dry-clean-only museum" lane by default (statement overlay dress may be the exception — document it).

## Relationship to Soft Structure (Jelly)

| | Soft Structure (jelly) | Soft Light (translucent garments) |
|---|---|---|
| Medium | TPU/PVC shoe | Sheer fabric clothing |
| Soft | Flexible translucent upper | Organza / mesh / chiffon / lace field |
| Hard | Molded arch, silver hardware | Lined zones, base layers, structure details |
| Exception | Breaks “patina over polish” (replaceable) | Usually does *not* break patina — fabric still wears in |
| Pairing | Completes warm-weather translucent *looks* | Clothing half of those looks |

They rhyme. They are not the same sub-line. A translucent outfit can use leather flats; a jelly day can use fully opaque clothes. The strongest Chaos Incorporated expression is when both registers meet **once** in a look (e.g. Ivory sheer sleeve + Emerald jelly flat).

## What This System Does Not Cover

- Intimate / sleepwear categories (out of wardrobe manifesto scope)  
- Sheer activewear as a performance category  
- Off-palette neon mesh, gold hardware on sheer pieces, unlined bust daywear

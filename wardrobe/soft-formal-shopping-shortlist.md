# Soft Formal Shopping Shortlist

*Event-layer formal. Doctrine: [`formal-system.md`](formal-system.md). **After Phase 1 Wave A** and ideally after soft day dress + lightweight blazer. Not Wave A competition.*

**Hard rules at try-on**
1. Sit, walk, climb a step — comfort still wins.  
2. Hem: knee–tea default; floor only with owned heel.  
3. **Silver hardware only** on bags, buttons, clasps.  
4. One large Emerald field max.  
5. No moto in the fitting room “formal” stack.

## Reuse before buying

| Already in system | Formal use |
|---|---|
| Ring statement dress | Path B Emerald ceremony |
| Anchor Heel | Long ceremonies, lower heel days |
| Heart-emerald pendant | Suit path or black/navy dress only |
| Wool wrap coat | Cold arrival instead of evening wrap |
| Layering silver set | Always |

## Do not buy for Soft Formal

| Skip | Why |
|---|---|
| Gold evening bag / gold buttons | Breaks Starlink One system |
| 4" stiletto-only | Comfort rule |
| Ballgown with cage | 64" + comfort fail |
| Hardware moto as formal shell | Wrong register (look 03 only) |
| Second Emerald formal dress same year | One loud field discipline |

# Sheer Long-Sleeve Base

**Category:** Tops  
**Subcategory:** Translucent — Sheer Long-Sleeve Base  
**Priority:** Experimental  
**Status:** Concept  

*Soft Light kit hero for layering. Governed by [`../../translucent-system.md`](../../translucent-system.md). Source first among translucent tops.*

## Why It Works for Her Frame (64")
- A close-fitting sheer long sleeve adds a second skin of color without adding bulk — proportion stays long rather than boxy.
- Proportion notes: body length hits high hip; sleeve **must clear the wrist** (end at or before the wrist bone) so Starlink One stays visible — same mandatory wrist rule as denim jacket and crew tee.
- Comfort priority: fine soft mesh or lightweight sheer knit, not scratchy costume mesh; seams flat or minimal against skin.

## Technical Specs
- Rise: N/A  
- Inseam / length: Body ~22–24"; sleeve to wrist or 3/4  
- Waist range / sizing notes: Slim fit; true to size or size for stretch recovery  
- Stretch level: 5–15% elastane in nylon/poly or cotton-blend sheer knit  
- Fabric weight: Very light, 60–100 gsm  
- Closure type: Pullover  

## Fabric & Feel
- Fabric composition: Soft power-mesh, fine gauge sheer jersey, or similar — **not** stiff tulle  
- Hand-feel / skin feel: Cool, smooth, barely-there; no rough mesh bite  
- Breathability: Excellent  
- Break-in behavior: Should feel correct immediately  

## Opacity map
- **Sleeves / arms:** translucent (primary light field)  
- **Body:** translucent — **must** be worn over opaque bra-friendly base or solid tank/cami for daywear  
- **Bust:** translucent fabric only when underlayer provides opacity

# The Barely There Sandal

*The warm-weather sandal, built as two versions — Everyday Strap and Dressy Slide — off one shared construction platform. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the Shenzhen-ready factory spec.*

The clearest warm-weather LoveShackFancy-DNA piece in the system, and the one shoe whose Starlink One connection is atmospheric (ribbon-detail motif) rather than a literal hardware reference. See [`../README.md`](../README.md) for its place in the full shoe catalogue.

## Status
Concept stage — full vision and manufacturing brief drafted, nothing sampled or physically confirmed yet. See `manufacturing-brief.md` Section 7 for the open items a factory needs to weigh in on before tooling — most notably whether the hybrid leather/fabric strap can be bonded at the intended narrow width — and Section 8 for the recommended first prototype scope (isolated strap swatch, then Everyday Strap in one size and colorway).

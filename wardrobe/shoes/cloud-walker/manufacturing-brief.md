# The Cloud Walker — Manufacturing Brief

**Document Type:** Technical Specification / Manufacturing Brief for factory quoting and prototyping
**Prepared for:** Footwear manufacturer, Shenzhen
**Revision:** 1.0 (Draft — pending factory confirmation of sole-height tolerance and Gore-Tex bootie sourcing)
**Governing doc:** [`vision.md`](vision.md) — read for design rationale; this document is the buildable spec.

**Status note, read first:** this brief translates an approved design direction into dimensions, materials, and construction methods for an initial quoting and prototyping conversation. Section 8 lists what needs factory engineering confirmation before tooling begins — the 1.25–1.5" sole-height target (Section 2) is the single most proportion-critical spec in this brief and should be confirmed against a physical sample before any production commitment.

---

## 1. Product Overview

A women's low-top sneaker with a cupsole construction, 1.25–1.5" total sole stack, visually layered midsole detail, soft leather or premium canvas upper, removable memory-foam insole, and an optional Gore-Tex waterproof bootie construction on select colorways.

## 2. Sole & Midsole Specification

| Parameter | Target | Tolerance |
|---|---|---|
| Total sole stack height | 32–38mm (1.25–1.5") | ±1.0mm — tight tolerance given how proportion-sensitive this figure is |
| Midsole material | Molded EVA or PU, cupsole construction | Confirm factory's standard cupsole compound and density |
| Midsole visual layering | Cosmetic tooling line dividing midsole into two visually distinct layers | Confirm tooling feasibility at the target stack height — this detail is what delivers the "chunky-inspired" read; a single uniform block reads flatter at the same height |
| Outsole | Rubber, moderate tread depth | Confirm standard tread pattern against reference silhouette |
| Toe shape | Rounded, moderate volume | Confirm last toe-box profile against reference, not a generic round-toe descriptor |
| Upper weight | 280–310gsm-equivalent | Confirm target hand-feel against physical swatch before bulk leather order |

## 3. Construction Method

- **Construction: cupsole.** The upper is cemented into a molded midsole/outsole unit. Confirm this is within the factory's standard capability — some factories specialize in vulcanized or cement-only sneaker construction and may need to subcontract or requote for cupsole tooling.
- **Layered midsole tooling (Section 2):** flag explicitly during the quoting call — this is a cosmetic-but-proportion-critical detail, not a structural one, and needs its own confirmation separate from the base cupsole capability question.
- **Gore-Tex bootie construction (select colorways):** a waterproof membrane bonded between the upper and interior lining, not a surface spray treatment. Confirm the factory has an existing relationship with a Gore-Tex licensed bootie supplier — this is a licensed-material sourcing question (similar in shape to a nib-housing or movement-family sourcing question in this brand's other technical briefs), not an open-market fabric purchase.

## 4. Insole / Footbed Build

| Layer | Spec |
|---|---|
| Sock lining | Soft textile or leather, matching upper material family |
| Insole | **Removable memory foam insole**, swappable for orthotics; confirm standard factory foam density and whether a branded/embossed removable insole is achievable at reasonable cost |
| Collar padding | Substantial padding at the heel collar specifically, to prevent heel rub — call out explicitly as a fit-quality requirement |

## 5. Materials

| Priority | Material | Spec | Used In |
|---|---|---|---|
| 1 | Soft leather | 280–310gsm-equivalent target hand-feel, standard sneaker-weight leather tannage | Flagship upper option |
| 2 | Premium canvas | Heavyweight canvas, no break-in period | Alternate upper option; primary pairing for the Gore-Tex bootie construction (Section 3) |

## 6. Starlink One Reference — Manufacturing Notes

- **Interior stitching:** hidden Starlink Emerald (`#0F5C3D`, see [`../../color-system.md`](../../color-system.md)) thread on the interior lining. Confirm thread color-match process against the factory's standard thread stock, same process as the Anchor Heel's heel-breast thread detail (see [`../anchor-heel/manufacturing-brief.md`](../anchor-heel/manufacturing-brief.md) Section 6).
- **Exterior accent colorway:** Starlink Emerald as sole, eyelet, or heel-tab accent detailing on select colorways — confirm which specific component the color applies to before finalizing colorway art, since "accent detailing" needs a specific component named for tooling/dyeing purposes rather than left open-ended.

## 7. Sizing & Last

- **Fit varies by upper material:** leather runs true to size, canvas sometimes runs large — confirm both explicitly with the factory and reflect the difference in size-guide documentation rather than assuming one sizing chart covers both upper options.
- **Last grading, not linear scaling:** independently grade US 5–7, same rule as every other shoe in this catalogue.

## 8. Critical Technical Challenges — Open Items for Factory Conversation

1. **Sole-height tolerance confirmation (Section 2)** — the single most proportion-critical spec in this brief; request a physical sample before finalizing production tooling.
2. **Cupsole capability confirmation** — gating requirement; confirm before other conversation items, since not all factories run this construction as standard.
3. **Layered midsole cosmetic tooling** — confirm feasibility at the target stack height separately from base cupsole capability.
4. **Gore-Tex bootie sourcing** — confirm existing licensed-supplier relationship; this is a sourcing/licensing question, not an open-market material purchase.
5. **Canvas-vs-leather sizing discrepancy** — get written confirmation of the fit difference so size guides can reflect it accurately.
6. **Thread color match for interior Starlink Emerald detail** — confirm against `../../color-system.md`'s hex reference.

## 9. Requested Next Step

A running prototype in the leather upper option only, single colorway (Cloud Ivory), single size (US 7), with the sole stack height and layered midsole tooling as the two specs to validate most carefully before any Gore-Tex, canvas, or multi-colorway expansion.

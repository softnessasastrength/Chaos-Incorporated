**Every item in the catalogue now has the full Anchor One treatment** — a `vision.md` design-rationale document plus a Shenzhen-ready `manufacturing-brief.md`, in its own subfolder. Items 1–8's original single-file concept notes (third column above) are left in place, unmodified in substance, since `full-looks/` and `bottoms/jeans/jeans-template.md` reference several of them directly by path — each now carries a one-line pointer to its fuller folder at the top. Items 9–13 are new silhouettes with no prior concept-note stage — they were designed directly into the full vision + manufacturing-brief format, the same way Anchor One itself was.

**Naming note:** items 1–4 are renamed/upgraded versions of the original Phase 1–2 shoes, not new duplicate items — original filenames and paths stayed stable so existing cross-references (`full-looks/`, `bottoms/jeans/jeans-template.md`, etc.) don't break. The catalogue's "Statement Mule" concept mapped onto two real, distinct silhouettes already in the system (a closed pump and an open-back mule) rather than one — both are listed under #6 rather than picking one arbitrarily.

**Design principles applied across all thirteen:** low-to-mid rise / stable base / no aggressive pitch for a 64" frame (with each heeled item's own honest accounting of where it sits against that rule — see the Kitten Heel Mary Jane and Lug-Sole Combat Boot in particular, both of which name a real tension rather than a clean fit); comfort non-negotiable (memory foam + real arch support); materials chosen to age like the ring, not resist aging — **except the Jelly Sandal**, the one deliberate, named exception to that rule, resolved by making the ring's own color the material itself rather than a surface treatment; every pair carries at least one Starlink One reference except the two deliberate function-first exceptions (Rain/Snow Boot, Yoga/Recovery Shoe) — placed hidden on everyday pieces, overt where genre convention expects hardware (Horsebit Loafer) or statement tier calls for it (Statement Mule), literal-as-substance on the Jelly Sandal, and quiet everywhere else. See each item's own vision document for the specific rule being applied.

Note from the bottoms system: several silhouettes (wide-leg jean, flare jean, barrel jean) explicitly depend on a specific shoe category (heeled boot, platform sole) to complete their proportion logic — cross-reference `../bottoms/jeans/jeans-template.md` when sourcing so shoe purchases match the bottoms they're engineered to pair with.

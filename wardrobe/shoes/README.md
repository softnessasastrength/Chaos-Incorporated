**Naming note:** items 1–4 are renamed/upgraded versions of the original Phase 1–2 shoes, not new duplicate items — filenames and paths stayed stable so existing cross-references (`full-looks/`, `bottoms/jeans/jeans-template.md`, etc.) don't break. The catalogue's "Statement Mule" concept mapped onto two real, distinct silhouettes already in the system (a closed pump and an open-back mule) rather than one — both are listed under #6 rather than picking one arbitrarily.

**Design principles applied across the core eight:** low-to-mid rise / stable base / no aggressive pitch for a 64" frame; comfort non-negotiable (memory foam + real arch support); materials chosen to age like the ring, not resist aging; every pair carries at least one Starlink One reference, placed hidden on everyday pieces and visible on statement pieces — see each item's own Starlink One Pairing section for the specific rule being applied.

**Jelly line exception:** [`jelly/`](jelly/) deliberately breaks the “patina over polish” material rule — wipeable, seasonal, replaceable TPU/PVC with molded arch support and silver-family chain-link hardware. Documented as controlled chaos, not a silent contradiction. Source the hero everyday jelly flat only after Phase 1 Must-Have shoes and the Barely There sandal are in motion; expand silhouettes only after a real warm-season trial.

Note from the bottoms system: several silhouettes (wide-leg jean, flare jean, barrel jean) explicitly depend on a specific shoe category (heeled boot, platform sole) to complete their proportion logic — cross-reference `../bottoms/jeans/jeans-template.md` when sourcing so shoe purchases match the bottoms they're engineered to pair with.

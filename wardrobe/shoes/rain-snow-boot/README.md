*The waterproof Chelsea boot — High tier, filling the one genuine functional gap the rest of the catalogue doesn't cover. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the Shenzhen-ready factory spec.*

The one shoe in the system where function fully overrides aesthetic signaling — no Starlink One reference, by deliberate design choice. See [`../README.md`](../README.md) for its place in the full 8-shoe catalogue.

# The Anchor Heel

*"Anchor One" — the everyday heel. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the Shenzhen-ready factory spec.*

Not to be confused with [`../06-block-heel-pump.md`](../06-block-heel-pump.md), the Phase 3 going-out statement heel — that piece is a rare-occasion bet at 2.5–3". This is the 2.0–2.25" Must-Have daily driver [`../README.md`](../README.md) already named as the better long-term investment.

Three variations, one shared last and construction method: **Everyday Anchor** (Must-Have), **Dress Anchor** (High), **Travel Anchor** (High).

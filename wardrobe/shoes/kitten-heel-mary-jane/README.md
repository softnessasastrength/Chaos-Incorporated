# The Kitten Heel Mary Jane

*The romantic strapped heel — Medium tier, bridging the Perfect Flat and the Anchor Heel. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the Shenzhen-ready factory spec.*

Names and resolves a real tension head-on: a kitten heel is thin by genre definition, which is exactly the profile the rest of the catalogue's heeled pieces avoid on stability grounds. See [`../README.md`](../README.md) for its place in the full shoe catalogue.

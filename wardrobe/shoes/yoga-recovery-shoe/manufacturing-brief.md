## 1. Product Overview

A women's loafer-silhouette recovery slip-on, 0–0.25" heel height, soft knit or brushed suede upper, substantial contoured arch support, 8–10mm memory foam footbed, elasticized topline entry.

## 2. Shoe Specification

| Parameter | Target | Tolerance |
|---|---|---|
| Heel height | 0–6mm (0–0.25") | ±0.5mm |
| Footbed cushioning | 8–10mm memory foam | ±0.5mm — the thickest footbed spec anywhere in this brand's footwear catalogue; confirm the factory's standard memory foam density (target 3–4 lb/ft³, per [`../anchor-heel/manufacturing-brief.md`](../anchor-heel/manufacturing-brief.md) Section 4) holds up at this increased thickness without excessive compression set over time |
| Arch support | Substantial, contoured, built into the footbed (not removable) | Confirm tooling for a more assertive arch contour than this brand's other footwear uses |
| Topline | Elasticized entry | Confirm elastic recovery spec — this component sees repeated stretch on every wear |

## 3. Knit Upper Construction — A Different Manufacturing Process

**Flag this explicitly and early:** a flat-knit or circular sock-knit upper is built on knitting machinery, not cut-and-sewn from leather panels — this is a genuinely different manufacturing process from every other shoe in this brand's catalogue, closer in kind to the Cloud Walker's Gore-Tex-bootie sourcing question (see [`../cloud-walker/manufacturing-brief.md`](../cloud-walker/manufacturing-brief.md) Section 3) than to a standard leather-goods build.

- Confirm whether the quoted factory has in-house knit-upper capability, or whether this requires a separate knit-manufacturing subcontractor relationship.
- Confirm minimum order quantities for custom knit patterns/colorways — knit uppers are typically programmed per style on dedicated machinery, and MOQs for a custom knit pattern can differ meaningfully from a cut-and-sew leather upper's MOQ.
- Get the brushed suede (Section 4, material option 2) quoted as a parallel path in case knit-upper sourcing proves difficult or costly at the target volumes — this option uses this brand's existing leather-goods manufacturing relationships rather than a new sourcing channel.

## 4. Materials

| Priority | Material | Construction | Notes |

| 1 | Soft knit | Flat-knit or circular sock-knit | Flagship, maximum comfort/breathability — see Section 3 for the sourcing implications |
| 2 | Brushed suede | Chrome or vegetable-tanned, natural nap | Uses this brand's existing leather-goods sourcing; recommended as the lower-sourcing-risk first prototype material if knit-upper sourcing (Section 3) needs more lead time to confirm |

## 5. Construction Method

- **Outsole:** flexible rubber, contoured specifically for natural foot movement — confirm this is a distinct outsole tooling from the stiffer, more structured outsoles used on this brand's heeled footwear; a stiff outsole here directly undercuts the recovery/comfort brief.
- **Footbed installation:** confirm whether the 8–10mm memory foam + contoured arch support is built as a fixed (non-removable) footbed unit, per the vision document's spec — this differs from the Cloud Walker's removable insole approach and should be explicitly distinguished in factory conversation so the two aren't conflated.

## 6. Sizing & Last

- **Room in the toe box, not a snug fit:** confirm this fit philosophy explicitly with the last-maker, since it's a deliberate departure from the "true to size, closer fit" approach used on the rest of the catalogue.
- **No break-in period expected:** confirm the first prototype sample feels correct immediately in wear-testing — a recovery shoe needing a break-in period is a signal of a construction problem, not normal wear-in behavior.
- **Last grading, not linear scaling:** same rule as the rest of the catalogue for US 5–7.

## 7. Critical Technical Challenges — Open Items for Factory Conversation

1. **Knit-upper sourcing (Section 3)** — confirm in-house capability vs. subcontractor relationship, and get suede quoted in parallel as a lower-risk fallback material.
2. **Memory foam compression set at increased 8–10mm thickness** — confirm the factory's standard foam holds up over time at a thicker-than-usual spec, not just at the 4–6mm spec used elsewhere in the catalogue.
3. **Elastic topline recovery over repeated wear** — confirm elastic material spec and expected service life.
4. **Fixed vs. removable footbed construction** — confirm this is built as specified (fixed unit) and not defaulted to a removable insole out of factory habit.

## 8. Requested Next Step

A running prototype in the brushed suede option first (lower sourcing risk, uses existing leather-goods manufacturing relationships), single size (US 7), single colorway (Cloud Ivory) — confirming the arch support and 8–10mm footbed spec feel correct in wear before pursuing the knit-upper sourcing conversation in parallel for the flagship material.

# Soft Formal System

*Design doctrine for rare-occasion formal wear in Renn's wardrobe. Governed by [`wardrobe-manifesto.md`](wardrobe-manifesto.md) and [`color-system.md`](color-system.md). Not everyday, not Phase 1 — event layer after foundation and elevated day pieces exist.*

**Register name:** Soft Formal  
**Priority for the register:** Medium / Experimental (never Must-Have — frequency is low)  
**Phase:** Phase 3-adjacent / event layer — **source after Phase 1 and after soft day dress + blazer exist**  
**Status:** Concept  

---

## Thesis

Chaos Incorporated dresses a body that lives in clothes, not a mannequin that visits events. Soft Formal is the answer when the event is real (wedding guest, cocktail, black-tie optional, solemn ceremony) and **moto-and-going-out is wrong**.

**Soft Formal = romantic structure for rare occasions, still engineered for sit / walk / dance / Starlink One.**

It is **not**:
- A second full wardrobe
- A ballgown cage
- A gold-jewelry exception
- The same as Soft Heat elevated day or Statement Going-Out (look 03)

---

## Occasion Map

| Occasion | Soft Formal answer | Reuse from system |
|---|---|---|
| Wedding guest (day) | Soft formal midi / tea dress + moderate heel | Soft day dress only if dress code is casual-wedding |
| Wedding guest (evening) / cocktail | Cocktail dress **or** soft formal suit | Ring statement dress if Emerald is wanted and dress code allows short formal |
| Black-tie optional | Emerald or black formal midi **or** soft formal suit + silk cami | Block pump; heart pendant only if dress is not already Emerald |
| Solemn (funeral, formal ceremony) | Midnight Navy or Jet Black crepe — quiet | Wool coat arrival in cold |
| True black-tie (rare) | One long formal dress **or** full formal suit — Experimental | Not both first |
| Chaos night out (not formal) | Stay on look 03 — ring dress + moto | Soft Formal wraps/coats, not moto |

## Non-Negotiable Rules

1. **Comfort still wins.** Sit test, walk test, bathroom logistics. No 4" stiletto-only. Prefer block / stacked / Anchor Heel.  
2. **64" hems:** default **knee to tea/midi**. Floor length only with explicit heel + hem plan (no pool on the ground).  
3. **Defined waist** on dresses and suits — volume needs an exit strategy.  
4. **Silver-family hardware and jewelry only** — no “formal gold.”  
5. **One large Emerald field** — dress *or* shoes *or* bag *or* pendant; ring always allowed as small saturated accent.  
6. **Wrist clear** when sleeves exist — Starlink One is part of formal jewelry, not an afterthought.  
7. **Re-wearable fabrics preferred** — crepe, stretch satin, soft suiting with recovery. Pure one-wear costume only as Experimental.  
8. **Opacity:** formal does not mean lingerie; lined bust and appropriate coverage for the event. Soft Light opacity map still applies if sheer is used.  
9. **Moto stays home** for Soft Formal looks — wrong register.

## Source Order

1. Phase 1 Must-Haves fitted  
2. Soft day dress + blazer (elevated day proven)  
3. **Soft formal midi** (highest re-wear among formal)  
4. Evening clutch + confirm block pump or Anchor Heel  
5. Cocktail dress **or** soft formal suit (pick by lifestyle, not both first)  
6. Evening wrap  
7. Long formal dress last / only if a real black-tie date exists  

---

## What Soft Formal Does Not Cover

- Bridal gown (separate project if ever needed)  
- Full menswear tux costume as default  
- Gold, crystal-on-gold, warm brass “evening” hardware  
- Weekly formal rotation (if she needs that, the dress code is wrong or the piece was mis-tiered)

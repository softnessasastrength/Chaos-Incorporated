# Straight-Leg Ponte Trouser

**Category:** Bottoms
**Subcategory:** Trousers — Straight-Leg Ponte (Comfort Trouser)
**Priority:** Medium
**Status:** Concept

## Why It Works for Her Frame (64")
- Straight leg reads clean without adding width, same logic as the straight-leg jean — the safest volume decision at this height.
- Proportion notes: hem at the top of the shoe, mid-to-high rise, same break point the rest of the bottoms system uses.
- Comfort priority: this is explicitly the "wears like leggings, looks like trousers" piece — the entire reason it exists in the system is to close the comfort gap the tailored wide-leg trouser can't fully close on a low-effort day that still needs to look put-together.

## Technical Specs
- Rise: Mid-to-high (9–10" front rise)
- Inseam / length: 27–28"
- Waist range / sizing notes: Elastic-backed waistband, true to size
- Stretch level: 15–20%, 4-way ponte stretch
- Fabric weight: Mid
- Closure type: Pull-on with hidden elastic waistband

## Fabric & Feel
- Fabric composition: Ponte knit — rayon/nylon/spandex blend
- Hand-feel / skin feel: Soft, substantial knit with structured drape but zero stiffness — the actual "comfort trouser" hand-feel this piece exists for
- Breathability: Moderate
- Break-in behavior: None needed — should feel correct from the first wear

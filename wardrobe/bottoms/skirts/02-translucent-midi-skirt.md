# Translucent Midi Skirt

**Category:** Bottoms  
**Subcategory:** Skirts — Translucent Midi  
**Priority:** Experimental  
**Status:** Concept  

*Soft Light bottoms piece. Governed by [`../../translucent-system.md`](../../translucent-system.md). Distinct from [`01-tiered-midi-skirt.md`](01-tiered-midi-skirt.md) (opaque romantic tiers).*

## Why It Works for Her Frame (64")
- Knee to just-below-knee length (same crop discipline as the tiered midi) so sheer volume doesn’t shorten the leg.
- Proportion notes: high waist with a **solid or fully lined yoke/waistband zone**; sheer field starts below that so the waist stays defined.
- Comfort priority: lightweight sheer over a soft slip or worn with bike shorts — waistband must not dig; no scratchy petticoat netting.

## Technical Specs
- Rise: High  
- Inseam / length: 20–24" (knee to just-below-knee)  
- Waist range / sizing notes: Elastic back or side zip; flat front preferred  
- Stretch level: Waistband only unless slip is stretch  
- Fabric weight: Shell very light 40–80 gsm; slip 80–120 gsm  
- Closure type: Side zip or pull-on with slip attached  

## Fabric & Feel
- Fabric composition: Soft chiffon / organza / fine mesh outer; opaque half-slip attached or designed as a set  
- Hand-feel / skin feel: Fluid outer; soft slip against legs  
- Breathability: Excellent  
- Break-in behavior: Minimal  

## Opacity map
- **Waistband / yoke:** opaque  
- **Seat / crotch:** covered by attached slip or mandatory bike-short underlayer (document which)  
- **Skirt field:** translucent — primary light field  
- **Default underlayer if slip not attached:** [`../shorts/03-bike-shorts.md`](../shorts/03-bike-shorts.md) in matching neutral

## Status
All three phases are built out — see [`build-plan.md`](build-plan.md) for the full breakdown and [`full-wardrobe.md`](full-wardrobe.md) for the complete catalog with priorities, statuses, and a Phase 1 shopping checklist. Phase 3 statement pieces were built ahead of the normal sequencing rule (Phase 1 complete, Phase 2 half-built per category) at explicit request; treat the Phase 1/2 pieces as the near-term shopping list and Phase 3 as the aspirational proof-of-concept. Soft Structure (jelly) and Soft Light (translucent garments) are Experimental seasonal registers documented early — source after core Must-Haves.

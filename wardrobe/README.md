# Renn Wardrobe

Full fashion design system: romantic, whimsical pieces (LoveShackFancy x Target DNA) engineered for a 63" frame, built around a fixed sterling-silver-and-emerald reference point — [Starlink One](../ring/README.md).

## Categories
- `/tops/` — `basics/` (quiet, high-rotation) + `intricate/` (puff sleeve, smocking, lace, ruffle)
- `/bottoms/` — jeans, leggings, shorts, trousers (see [`bottoms/bottoms-priority.md`](bottoms/bottoms-priority.md)) — most built-out category so far
- `/outerwear/` — structured, soft, and romantic layers
- `/shoes/` — flats, sneakers, boots, heels, sandals
- `/accessories/` — jewelry, hair, belts, bags, scarves — heaviest direct Starlink One integration
- `/full-looks/` — assembled outfits spanning categories, built once enough pieces exist

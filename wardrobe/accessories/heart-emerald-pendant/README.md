# The Heart-Emerald Pendant

*The fine-jewelry companion piece to Starlink One — Experimental tier (Phase 3 Statement), and the accessories category's most literal single-object echo of the ring. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the jeweler-ready commission spec.*

The one deliberate exception to the wardrobe's "hidden, not announced" rule for Starlink One references — every other reference in the system (the Anchor Heel's debossed insole mark, the Jelly Sandal's translucent color-as-material) is designed to be discovered; this one is designed to be recognized immediately. See [`../README.md`](../README.md) for its place in the accessories catalogue and the original concept note at [`../06-heart-emerald-pendant-necklace.md`](../06-heart-emerald-pendant-necklace.md), which this folder expands into full vision + manufacturing-brief format.

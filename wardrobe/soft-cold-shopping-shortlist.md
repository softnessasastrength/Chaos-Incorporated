# Soft Cold Shopping Shortlist

*Cool/cold capsule. Doctrine: [`cold-season-system.md`](cold-season-system.md). **After Phase 1 Wave A** (jean + tee + bag). Prefer Hug Boot path before full winter coat if climate is mild.*

**Hard rules at try-on**
1. Coat mid-thigh to knee on 64" — reject floor-length defaults.  
2. Sleeve push-up clears wrist for Starlink One.  
3. Silver buttons/toggles/snaps only.  
4. Rain/snow boot: real lug tread + waterproof seams (function over fashion).  
5. No warm camel/gold — cool greige only if in Sterling Grey family.

## Climate shortcuts

| Climate | Buy |
|---|---|
| Mild winter | C1 only + trench if rain |
| Real winter | C1 → C2 → C3 coat |
| Ice/snow city | Prioritize rain/snow boot in C2 before fashion knee-high |

# The Sheer Overlay Dress

*The Soft Light kit's statement dress — Experimental tier (Phase 3 Statement-adjacent). Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the pattern-and-fit-ready manufacturing spec.*

Two-part construction — opaque/solid underdress plus a fully sheer chiffon or organza overlay — governed by [`../../translucent-system.md`](../../translucent-system.md)'s opacity map. Distinct from [`../ring-statement-dress/`](../ring-statement-dress/) (opaque Emerald showcase, not translucent). **Built last** in the translucent kit, after the lower-stakes Soft Light pieces prove the opacity map in real wear. See [`../README.md`](../README.md) for its place in the dresses category and the original concept note at [`../02-sheer-overlay-dress.md`](../02-sheer-overlay-dress.md), which this folder expands into full vision + manufacturing-brief format.

## Status
Concept stage — full vision and manufacturing brief drafted, nothing sampled or opacity-tested yet. See `manufacturing-brief.md` Section 3 for the gating requirement — the underlayer must pass opacity testing under both direct sunlight and event lighting before any production commitment — and Section 8 for the recommended first step (a single Cloud Ivory/Cloud Ivory opacity-tested sample).

# The Soft Formal Midi Dress

*The Soft Formal kit's workhorse — High tier within Soft Formal, recommended first purchase in the formal kit. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the pattern-and-fit-ready manufacturing spec.*

Doctrine: [`../../formal-system.md`](../../formal-system.md). Distinct from [`../soft-day-dress/`](../soft-day-dress/) (casual romantic) and [`../ring-statement-dress/`](../ring-statement-dress/) (Emerald showcase going-out). Tea-to-midi length threads the needle between too-short (cocktail register) and floor-length (requires the heel-first sequencing the Long Formal Dress needs). See [`../README.md`](../README.md) for its place in the dresses category and the original concept note at [`../04-soft-formal-midi-dress.md`](../04-soft-formal-midi-dress.md), which this folder expands into full vision + manufacturing-brief format.

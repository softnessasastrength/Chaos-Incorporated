## Why It Works for Her Frame (64")
- Two-part design: opaque or solid **underdress / slip dress** + sheer **overlay** (or integrated sheer outer with full opaque inner). Length hits midi at knee-to-mid-calf, not floor, unless underlayer hem is intentional.
- Proportion notes: defined waist on the underlayer (smocked, seamed, or slip with waist clip); sheer outer can float without erasing the waist line.
- Comfort priority: soft inner against skin all day; sheer outer never the only layer on bust/seat.

## Technical Specs
- Rise: N/A (dress)  
- Inseam / length: Underlayer knee to midi; overlay same or 1–2" longer for float  
- Waist range / sizing notes: True to size on underlayer; overlay one ease size OK  
- Stretch level: Underlayer 2–5% preferred; overlay 0%  
- Fabric weight: Underlayer 120–180 gsm; overlay 30–70 gsm  
- Closure type: Underlayer zip or pull-on; overlay open or shared closure  

## Fabric & Feel
- Fabric composition: Stretch crepe / satin / ponte underlayer + chiffon or organza overlay  
- Hand-feel / skin feel: Soft inner; airy outer  
- Breathability: Good  
- Break-in behavior: Minimal  

## Opacity map
- **Underlayer:** fully opaque at bust, seat, crotch — **part of the design, not optional**  
- **Overlay:** fully translucent  
- If sold as sheer-only without underlayer, **out of system** unless a specific matching underlayer is sourced as a set

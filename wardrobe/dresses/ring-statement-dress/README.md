# The Ring Statement Dress

*The single piece in the entire system built explicitly to showcase Starlink One — Experimental tier (Phase 3 Statement). Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the pattern-and-fit-ready manufacturing spec.*

Named in [`../../build-plan.md`](../../build-plan.md) as "the piece that most literally executes the wardrobe is designed around the ring" — every design decision (fitted bodice, knee-length hem, 3/4-to-cropped sleeve) exists to keep Starlink One visible through natural movement, not just a static photograph. See [`../README.md`](../README.md) for its place in the dresses category and the original concept note at [`../01-ring-statement-dress.md`](../01-ring-statement-dress.md), which this folder expands into full vision + manufacturing-brief format.

# Dresses

A sixth category, added during Phase 3 — not in the original five-category architecture, because a dress replaces both a top and a bottom at once and doesn't cleanly belong filed under `tops/`. See the update note in [`../wardrobe-architecture.md`](../wardrobe-architecture.md).

Phase 3 statement piece built: [`01-ring-statement-dress.md`](01-ring-statement-dress.md) — the wardrobe's single piece designed explicitly to showcase Starlink One, per [`../build-plan.md`](../build-plan.md)'s Phase 3 scope.

This category stays intentionally small — occasion dressing, not everyday rotation. Expand opportunistically, not on a schedule.

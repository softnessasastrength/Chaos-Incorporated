# The Long Formal Dress

*The true black-tie / floor-length path — Experimental tier, and the one dress in the catalogue with a hard sourcing gate attached to it. Read [`vision.md`](vision.md) first, then [`manufacturing-brief.md`](manufacturing-brief.md) for the tailoring-ready manufacturing spec.*

**Do not source without a real calendar event and an already-owned formal heel** — floor length at 64" only works if heel height is fixed first and the hem is tailored after, never the reverse. See `vision.md` Section 7 for the full sourcing sequence. Doctrine: [`../../formal-system.md`](../../formal-system.md). See [`../README.md`](../README.md) for its place in the dresses category and the original concept note at [`../06-long-formal-dress.md`](../06-long-formal-dress.md), which this folder expands into full vision + manufacturing-brief format.

# Full Look: Black-Tie Optional Suit

**Occasion:** Black-tie optional, formal work-evening, non-dress wedding guest.  
**Phase:** Soft Formal  
**Registers:** matching soft formal suit — hard architecture path without a dress.

## The Pieces
- **Suit:** [`../outerwear/10-soft-formal-suit.md`](../outerwear/10-soft-formal-suit.md) — Jet Black or Midnight Navy  
- **Under layer:** [`../tops/intricate/09-ribbon-tie-camisole.md`](../tops/intricate/09-ribbon-tie-camisole.md) — Cloud Ivory **or** [`../tops/translucent/03-lace-panel-camisole.md`](../tops/translucent/03-lace-panel-camisole.md) — Ivory (lined bust)

- **Bag:** [`../accessories/09-evening-clutch.md`](../accessories/09-evening-clutch.md) — Jet Black or pewter-silver  
- **Jewelry:** Starlink One + [`../accessories/02-layering-necklace-bracelet-set.md`](../accessories/02-layering-necklace-bracelet-set.md); optional [`../accessories/06-heart-emerald-pendant-necklace.md`](../accessories/06-heart-emerald-pendant-necklace.md) — **yes here**, suit is quiet so pendant can be the collarbone emerald echo  

## Why This Combination Works
- **Non-dress formal** without leaving the system for a random tux costume.
- **Soft + hard:** soft cami, hard matching suit structure, stable block heel.
- **Emerald allowed at pendant scale** because clothing field is navy/black — one loud green at jewelry only (ring + pendant both emerald-family is OK if pendant is small; skip pendant if it feels busy next to Starlink One).
- **64" proportion:** short blazer + heeled trouser break.

## Starlink One
- Blazer cuff clear; silver buttons only.
- Pendant optional — test in mirror with ring so they match, don’t compete.

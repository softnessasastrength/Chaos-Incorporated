# Full Look: Structured Everyday

**Occasion:** Everyday — errands, casual daytime, low-effort but intentional.
**Phase:** 1 (Essentials) — every piece below is Must-Have tier.
**Registers the manifesto's "soft materials, deliberate structure" rule via denim-on-denim done in contrasting washes, not matching ones.**

## The Pieces
- **Bottom:** [`../bottoms/jeans/01-straight-leg-jean.md`](../bottoms/jeans/01-straight-leg-jean.md) — Classic mid-blue indigo
- **Top:** [`../tops/basics/01-fitted-crew-tee.md`](../tops/basics/01-fitted-crew-tee.md) — White, tucked
- **Outerwear:** [`../outerwear/01-cropped-denim-jacket.md`](../outerwear/01-cropped-denim-jacket.md) — **Black denim colorway specifically**, not the mid-blue indigo option — deliberately contrasts against the mid-blue jean so the look doesn't read as an accidental matching denim set

## Why This Combination Works
- **Silhouette:** straight-leg jean + cropped jacket keeps the vertical line unbroken at the waist — nothing here fights the leg-lengthening the jean is built for.
- **Wash logic:** black jacket over mid-blue jean avoids the tonal-matching-denim problem both item files flag independently — this is the pairing they were each written to expect.
- **Hardware:** every silver-tone spec across the jacket, boot zip pull, bag, and jewelry is consistent — no gold anywhere in the look, per the closed hardware rule in `color-system.md`.
- **Color system:** white, black, and denim indigo are all on-palette (Jet Black / neutral); the look is intentionally quiet on color so the ring stays the only saturated-color note in the outfit.

## Starlink One
- Both the denim jacket and the crew tee are hemmed/sized so the wrist stays fully visible — this is the look the jacket's mandatory sleeve-length spec exists for.
- The layering bracelet sits on the same wrist as the ring; chain-link scale should be checked side by side per the jewelry item's own note, so the two read as a matched pair rather than clutter.
- Net result: the ring is the single point of color/detail interest in an otherwise deliberately quiet, structured outfit — exactly the "recede so the ring leads" logic named throughout the bottoms and outerwear notes.

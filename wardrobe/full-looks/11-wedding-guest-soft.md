# Full Look: Wedding Guest Soft

**Occasion:** Day or early evening wedding guest, garden formal, soft ceremony.  
**Phase:** Soft Formal  
**Registers:** Soft Formal workhorse midi — not Chaos moto night.

## The Pieces
- **Dress:** [`../dresses/04-soft-formal-midi-dress.md`](../dresses/04-soft-formal-midi-dress.md) — Blush Mauve, Cloud Ivory, or Sage  
- **Shell (optional mild):** [`../outerwear/11-evening-wrap-coat.md`](../outerwear/11-evening-wrap-coat.md) — Cloud Ivory or Sterling Grey  
- **Shell (cold):** [`../outerwear/09-wool-wrap-coat.md`](../outerwear/09-wool-wrap-coat.md) — Midnight Navy or Sterling Grey  
- **Shoes:** [`../shoes/anchor-heel/`](../shoes/anchor-heel/) **or** [`../shoes/06-block-heel-pump.md`](../shoes/06-block-heel-pump.md) — Cloud Ivory or Jet Black  
- **Bag:** [`../accessories/09-evening-clutch.md`](../accessories/09-evening-clutch.md) — Cloud Ivory or Blush Mauve  
- **Jewelry:** Starlink One + [`../accessories/02-layering-necklace-bracelet-set.md`](../accessories/02-layering-necklace-bracelet-set.md)  
- **Optional:** [`../accessories/04-hair-ribbons-scrunchies.md`](../accessories/04-hair-ribbons-scrunchies.md) only if dress code is soft/romantic — skip for strict formal  

## Why This Combination Works
- **Ceremony-correct** without ballgown volume; midi + moderate heel is 64"-safe.
- **Quiet romantic palette** lets the ring lead; no Emerald dress + Emerald bag.
- **No moto** — evening wrap or wool coat only.
- **Soft + hard:** soft crepe dress, structured waist, stable heel architecture.

## Starlink One
- Sleeves clear wrist; silver layering only.
- Heart pendant optional only if dress is Ivory/Blush and event wants more jewelry — not required.

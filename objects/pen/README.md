# Signature Pen

"Gallery One" — the over-engineered personal pen for the Chaos Incorporated universe. A piston-fill fountain pen with an open demonstrator window over the ink chamber and piston, the same "soft thing held by visible hard architecture" argument as [Starlink One](../../ring/README.md) and [Starlink Watch](../../watch/README.md), run through ink instead of a stone or a gear train.

Start here:
1. [`vision.md`](vision.md) — the foundational vision document covering format, materials, the ink window, nib, filling mechanism, and the hidden Starlink One references
2. [`manufacturing-brief.md`](manufacturing-brief.md) — the Taiwan-ready factory spec, dimensions, tolerances, and sourcing notes derived from the vision document

## Status
Concept stage — full vision and manufacturing brief drafted, nothing sampled, sourced, or physically confirmed yet. See `manufacturing-brief.md` Section 11 for the highest-priority open question (whether unprotected sterling silver holds up gracefully against real ink contact, which needs an answer before the flagship material option is locked) and Section 13 for the recommended first prototype scope (Option B, brass/rhodium-plated, functional window/nib/piston only).

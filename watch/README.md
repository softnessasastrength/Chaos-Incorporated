# Starlink Watch

A mechanical watch for the Chaos Incorporated universe — Apple Watch silhouette and proportions, real hand-wound movement, open dial and caseback, designed to be worn on the same wrist as [Starlink One](../ring/README.md) without competing with it.

Start here: [`starlink-watch-vision.md`](starlink-watch-vision.md) — the foundational vision document covering case, dial, movement, hands, crown, color, and strap system.

## Status
Concept stage — foundational vision only. No CAD, no movement sourcing, no factory engagement yet. See the vision doc's own "Open Questions for Next Steps" section for what has to be resolved before this moves toward a technical/manufacturing spec (the next step, once this vision is confirmed, would mirror `../wardrobe/` and `../packaging/`'s pattern of a dedicated `technical-specs.md`).
